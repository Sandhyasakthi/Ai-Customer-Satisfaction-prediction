AI-Driven Customer Satisfaction Prediction System
An AI-powered web application that predicts customer satisfaction based on service-related inputs.
The system uses a machine learning model integrated with a Flask web application to analyze customer feedback parameters and generate satisfaction predictions.

Project Overview
Customer satisfaction is a key factor in improving business services. This project demonstrates how machine learning can be used to predict customer satisfaction levels using service quality indicators.

The application allows users to:

Register and login to the system
Enter service-related input parameters
Predict customer satisfaction using a trained ML model
Store prediction results in a database

Machine Learning Model
The system uses a CatBoost Classifier model trained on customer service related features.

Example features used:

Service quality rating
Product satisfaction
Delivery experience
Customer support rating
The model predicts whether the customer is:

Satisfied
Not Satisfied
Installation and Setup
Clone Repository
git clone https://github.com/your-username/tata-steel-ai-project.git

Navigate to Project Folder
cd tata-steel-ai-project

Install Dependencies
pip install -r requirements.txt

Run Application
python app.py

Application Workflow
User creates an account
User logs into the system
User accesses the dashboard
User enters service related inputs
System processes the inputs through the ML model
Customer satisfaction prediction is generated
Prediction result is stored in the database
